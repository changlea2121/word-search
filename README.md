# Word Search & Inverted Index

This program reads a text file, builds an inverted index of all words, 
and lets you search for any word to find which lines it appears on.

## Features
- Reads any plain text file
- Cleans punctuation and lowercases all words
- Builds an inverted index mapping each word to the lines it appears on
- Interactive search: enter a word to find its line numbers

## Requirements
- python 3.x

## Usage
1. Clone this repository
2. Add your text file as `text.txt` in the same folder
3. Run the program:`python search.py`
4. Enter a word to search, or type `quit` to exit
